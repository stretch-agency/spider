# Stretch Agency Spider
![NPM Version](https://img.shields.io/npm/v/stretch-spider)

### [Web Documentation](https://stretch-spider.netlify.app/)

## Installation
Add the spider to your project by running
```
yarn add stretch-spider
```
or
```
npm install stretch-spider --save
```

After that you can import it to your css
```
@import '~/stretch-spider'
```
or your html
```
<link rel="stylesheet" href="./node_modules/stretch-spider/index.css">
```
